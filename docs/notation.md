# 主要符号表

## 基本数学符号

| 符号 | 含义 |
|:---:|:---|
| $x$ | 标量 |
| $\boldsymbol{x}$ | 向量 |
| $A$ | 矩阵 |
| $I$ | 单位阵 |
| $(·,·,·)$ | 行向量 |
| $(;,;,)$ | 列向量 |
| $(·)^T$ | 向量或矩阵转置 |

## 集合与空间

| 符号 | 含义 |
|:---:|:---|
| $\mathbb{R}$ | 实数集 |
| $\mathbb{R}^+$ | 正实数集 |
| $\mathcal{X}$ | 样本空间或状态空间 |
| $\mathcal{H}$ | 假设空间 |
| $\{\cdots\}$ | 集合 |
| $[m]$ | 集合 $\{1,\dots,m\}$ |
| $\lvert\{\cdots\}\rvert$ | 集合 $\{\cdots\}$ 中元素的个数 |

## 概率与统计

| 符号 | 含义 |
|:---:|:---|
| $\mathcal{D}$ | 概率分布 |
| $D$ | 数据样本（数据集） |
| $P(\cdot)$, $P(\cdot\mid\cdot)$ | 概率质量函数，条件概率质量函数 |
| $p(\cdot)$, $p(\cdot\mid\cdot)$ | 概率密度函数，条件概率密度函数 |
| $E_{\cdot\sim\mathcal{D}}[f(\cdot)]$ | 函数 $f(\cdot)$ 对 $\cdot$ 在分布 $\mathcal{D}$ 下的数学期望；意义明确时将省略 $\mathcal{D}$ 和（或）$\cdot$ |

## 函数与运算

| 符号 | 含义 |
|:---:|:---|
| $\mathfrak{L}$ | 学习算法 |
| $\lVert\cdot\rVert_p$ | 范数，$p$ 缺省时为 $L_2$ 范数 |
| $\sup(\cdot)$ | 上确界 |
| $\inf(\cdot)$ | 下确界 |
| $\mathbb{I}(\cdot)$ | 指示函数，在 $\cdot$ 为真和假时分别取值为 $1$、$0$ |
| $\text{sign}(x)$ | 符号函数：$x<0$ 时为 $-1$，$x=0$ 时为 $0$，$x>0$ 时为 $1$ |
